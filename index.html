Npm Modules to vanila js

How can I use my nodejs modules in a vanilla javascript file for frontend scripting?

What is npm

Npm package manager is a tool which will allow you to install third party libraries (other people's code) by using the command line. In node app.

What is node?

Node. js runtime is basically what will understand your javascript code and execute it to produce a result.

Differences between Node and the Browser

Both the browser and Node use JavaScript as their programming language.

With the browser javascript we can done only front end,..

Node apps brings with it, a huge advantage - the comfort of programming everything, the frontend and the backend, in a single language.

In the browser, most of the time what you are doing is interacting with the DOM, or other Web Platform APIs like Cookies. Those do not exist in Node, of course. You don’t have the document, window and all the other objects that are provided by the browser.

And in the browser, we don’t have all the nice APIs that Node.js provides through its modules, like the filesystem access functionality.

What is vanila js

VanillaJS is a name to refer to using plain JavaScript without any additional libraries like jQuery.

What is Babel ?

Babel is a free and open-source JavaScript transcompiler that is mainly used to convert ECMAScript 2015+ code into a backwards compatible version of JavaScript that can be run by older JavaScript engines.

What is Browserify ?

Why npm to js?

Occasionally, we need to use a JS library in our browser side, yet it’s only available from the NPM repository.

npm init and npm install

npm init is a convenient way of deliver your package. json,you may need to run it everytime you are starting a new project.

npm install , however, installs your dependencies in node_modules folder. You may need to run this everytime you manually add a dependency to your package.
